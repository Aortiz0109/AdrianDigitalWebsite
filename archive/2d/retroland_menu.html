Retroland Main Menu

Pixel Art | 2024

This was a brief commission to design menu art for a retro game being developed. I'll update with links to support it once it's released. The concept was centered around an arcade cabinet in the center of the frame that the player interacts with to begin playing. Originally, I set the scene in an arcade; however, the scene felt too static, so I made a fun Tron/Space scene to put the game cabinet in. There are also various spritesheets created for background animations that will be in the final product.

Once the scene was composed, I decided to create two different color schemes to give the client options on what direction they would like to go in. The client liked both and couldn't choose, so they decided to use both and make it randomized when you open the game. It added some extra workload animating things in two colors; however, it was nice to use both color schemes since I also enjoyed them.

As some bonus content for this page, I also included a character and grenade design for the same client for a game that has been put on hiatus. I didn't want to create a separate page for this; however, it's sad to see these assets go unused and showcases more of my pixel art.
